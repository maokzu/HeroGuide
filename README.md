# HeroGuide

**HeroGuide** est un addon World of Warcraft conçu pour simplifier la réalisation de hauts-faits complexes (ex: "Gloire au héros", méta-hauts-faits de zone).

![Version](https://img.shields.io/badge/version-v0.1-blue)

## Fonctionnalités

*   **Structure en Arbre (Graph View)** : Visualisez clairement la hiérarchie des méta-hauts-faits. Fini les listes plates illisibles !
*   **Intégration AllTheThings (ATT)** : Utilise la puissante base de données d'ATT pour vous dire exactement quoi faire.
*   **GPS Intégré** : Cliquez sur un objectif pour obtenir instantanément une flèche TomTom vers le PNJ, l'objet ou le lieu.
*   **Lazy Loading** : Performance optimisée. Les données ne sont chargées que lorsque vous explorez une branche.
*   **Suivi en Temps Réel** : L'interface se met à jour automatiquement quand vous validez un critère.

## Prérequis

*   **AllTheThings** (Obligatoire) : Nécessaire pour les données.
*   **TomTom** (Recommandé) : Pour les flèches GPS.

## Installation

1.  Téléchargez la dernière "Release" depuis la page GitHub.
2.  Décompressez le dossier dans `World of Warcraft/_retail_/Interface/AddOns/`.
3.  Assurez-vous que le dossier se nomme `HeroGuide`.

## Utilisation

*   L'addon ajoute un bouton sur la Minimap.
*   Cliquez pour ouvrir l'arbre.
*   Traquez un haut-fait dans l'interface de base de WoW pour le charger automatiquement dans HeroGuide.
